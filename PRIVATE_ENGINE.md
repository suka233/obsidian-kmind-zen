# Private KMind Zen Engine Boundary

This public source snapshot intentionally excludes the proprietary KMind Zen shared engine packages used by the installable Obsidian release asset.

Excluded packages include:

- '@kmind/core'
- '@kmind/app'
- '@kmind/editor-react'
- '@kmind/app-react'
- '@kmind/icons'
- '@kmind/i18n'

The public snapshot is intended to let reviewers inspect the Obsidian host integration surface while preserving KMind Zen's private engine source.
